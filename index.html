第一篇博客-hexo的安装 

这是我第一篇博客，也不在乎是不是markdown的格式，以后再慢慢改进嘛。
这片被博客记录我如何安装hexo博客。其实我主要参考了两个人的教程。

https://www.jianshu.com/p/e132ee632dec 这篇博客
https://www.bilibili.com/video/av44544186 程序羊，羊哥的视频讲解

第一步
下载nodejs
去官网下载nodejs（https://nodejs.org），下载完安装在哪儿就随你心意了.
最后成功的的截图如下![enter description here](./images/nodejs.png)


第二步
下载cnmp
在cmd（最好以管理员身份打开），我是win10的系统用的是windows powershell
输入命令 npm install -g cnpm --registry=https://registry.npm.taobao.org.
这一步，是国外的资源访问太慢，所以采用国内的镜像文件
成功的截图如下![enter description here](./images/cnpm.png)

第三步
下载hexo
在windows powershell中输入命令 cnpm install -g hexo-cli

第四步
开始搭建博客
在你想要的位置创建目录，以后你的文件基本都在这儿了
输入hexo init初始化，成功的截图如下
![enter description here](./images/XHAF$U9A@T9KMO96)TO8V@A.png)
因为我博客搭建好了，所以会多出一两个文件。

至此博客的搭建已经全部完成了，下载需要做的是，把他部署到远程的服务机上去.采用
github仓库的方式白嫖.

之后的方式就是对github的操作了。
我说一下我遇到的小问题
第一个，没有编辑md文件的编辑器，于是我下了小书匠，其实可以找找看其他的.
第二个, git的初始化,git一定要初始化！！！不初始化不能用
git config --global ....啥啥啥的，这是在命令行里面使用，而不是让你编辑到config.md文件中去！！
第三个，这个格式的文件有点特殊，注意空格！
